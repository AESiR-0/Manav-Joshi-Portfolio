The easy way

Let your AI do it

  1. Download the folder.
  2. Open it in Claude Code or Cursor.
  3. Say "set up my Kaka".

It carries a runbook and does the rest. You just click one button and pick a password.

The hands-on way

Six little commands

# free Cloudflare + Gemini key, then
npm install
npx wrangler login
npx wrangler kv namespace create KAKA_KV
npm run build && npx wrangler deploy
npx wrangler secret put SITE_PASSWORD
npx wrangler secret put GEMINI_API_KEY
