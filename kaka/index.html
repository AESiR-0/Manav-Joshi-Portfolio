What it does

Smart to-dos and habits.

✦

Speak, don't fill forms. One sentence in, a scheduled task out.

✦

Three things each morning. Picked before you wake up. One tap accepts.

✦

Pillars, not categories. A chip lights up when you feed an area of your life.

✦

Reminders that reach you. Push notifications, on by default for every dated task.

✦

A weekly review with opinions. Six honest sentences every Monday.

✦

It files your life. Meals, workouts, learnings. Said once, filed right.

Setup

Ten minutes, once.

The 2026 way

Let your AI do it

  1. Download the folder.
  2. Open it in Claude Code or Cursor.
  3. Say "set up my Kaka".

The repo carries the runbook. Your agent stops only where a human is required: one Allow click, two secrets typed by you.

The manual way

Six commands

# free Cloudflare account + free Gemini key
npm install
npx wrangler login
npx wrangler kv namespace create KAKA_KV
npm run build && npx wrangler deploy
npx wrangler secret put SITE_PASSWORD
npx wrangler secret put GEMINI_API_KEY
